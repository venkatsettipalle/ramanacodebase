# Registration and Login Example with Spring Security, Spring Boot, Spring Data JPA, HSQL, JSP

## Guide
https://github.com/venkatsettipalle/ramanacodebase.git

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP
- HSQL

## Run
```mvn clean spring-boot:run```
